### 🫀 VitalWatch — Non-Invasive Health Monitor
> *"What if a wearable could replace a blood test?"*

A clinical-grade embedded health monitoring system that estimates **blood glucose** and **blood pressure** non-invasively using a near-infrared optical sensor — no needles, no lab visit.

**⚙️ Engineering Challenge**  
Conditioning raw, noise-heavy NIR photoplethysmographic signals before they ever reach the ML layer — and keeping the entire inference pipeline fast enough to run at the edge.

```
Embedded C · I²C Protocol · ESP32
Python · XGBoost · Signal Processing · NHANES Dataset
```

🏆 **2× Hackathon Winner** &nbsp;·&nbsp; 📄 **Patent Filed**

### 🚗 Automotive Telematics Node
> *"Edge telemetry that runs for days on a battery."*

A power-optimized IoT firmware node on ESP32 that streams real-time vehicle gas-sensor telemetry to a cloud dashboard — with **deep-sleep + wake-on-interrupt** modes cutting active power draw to a fraction of baseline.

**⚙️ Engineering Challenge**  
Designing firmware that survives volatile automotive supply conditions while staying cloud-connected, with deterministic wake-and-transmit timing under interrupt control.

```
ESP32 · Interrupt-Driven Firmware · Deep-Sleep
Wi-Fi / HTTP · MATLAB ThingSpeak · Low-Power Design
```

### 🚦 AI Traffic Controller on FPGA
> *"Traffic lights that adapt to congestion in real-time."*

An FPGA-accelerated traffic control system featuring multi-FSM architecture for signal sequencing, timer control, and fail-safe emergency override — with an ML model classifying congestion levels and sending commands over UART.

**⚙️ Engineering Challenge**  
Maintaining deterministic FSM timing guarantees in Verilog while integrating a non-deterministic ML classification pipeline via asynchronous serial communication — bridging the discrete and probabilistic worlds.

```
Verilog HDL · Xilinx FPGA · Multi-FSM Design
Python ML · UART Interface · Timing Analysis
```

🏆 **14th Rank — VLSI Hackathon, NIT Kurukshetra**
